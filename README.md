## Installation

1. Clone repo first.

```bash
  
  git clone https://github.com/muxair080/multi_agent_email_response_generator.git
```

2. Create VENV
```bash
  python3 -m venv venv
  source venv/bin/activate
```

3. Requirments Installation
```bash
  pip install -r requirements.txt

```
3. Run Streamlit app
```bash
  Streamlit run Streamlit_app.py

```
